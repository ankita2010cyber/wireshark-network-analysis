# Network Traffic Analysis Using Wireshark

## Objective

This project demonstrates beginner-level Digital Forensics and Incident Response (DFIR) skills using Wireshark for packet capture and network traffic investigation.

## Tools Used

* Wireshark
* Windows

## Investigation Performed

* Packet capture and traffic monitoring
* DNS request analysis
* TLS encrypted communication analysis
* Endpoint activity investigation
* Protocol filtering and packet inspection

## Findings

The investigation identified DNS requests, encrypted TLS traffic, and communication between internal and external endpoints to understand network behavior and security monitoring workflows.

## Skills Gained

* Packet Analysis
* Network Monitoring
* Traffic Investigation
* Digital Forensics Basics
* Incident Response Fundamentals
